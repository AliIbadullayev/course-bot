# course-bot

Course bot for performing laboratory works for "Computer Architecture" discipline.

## Usage

`docker run -e BOT_TOKEN=... -v $HOME/course-data:/data --restart=on-failure:10 --name csa-bot ghcr.io/ryukzak/course-bot:master`
